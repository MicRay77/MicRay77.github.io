## ✍️ Text Formatting
**Bold**        →  `**Bold**`  
*Italic*        →  `*Italic*`  
~~Strikethrough~~ → `~~Strikethrough~~`  
`Inline code`   →  `` `Inline code` ``  

---

## 📋 Lists

### Unordered
- Item
- Item
  - Nested Item
* Item

### Ordered
1. First
2. Second
   1. Sub-second

## ✅ Task Lists
- [x] Completed Task
- [ ] Incomplete Task
